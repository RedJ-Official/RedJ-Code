Export

Allows you to export the file with syntax highlighting as HTML or RTF document.

Export Type

  • HTML: Export as HyperText.
  • RTF: Export as Rich Text Format.

Export Range

  • Whole Document: Export all text.
  • Selection Only: Export the selected text only.

Export Options

  • Include Line Numbers: If checked, includes line numbers in the exported file.
  • Use Original Font: If checked, saves the file using the editor font. If not, no font information is saved.
  • Use <PRE> Tag: For HTML exports: If checked, saves the text content in a <PRE>-Tag.
  • Use <STYLE> Tag: For HTML exports: If checked, saves the style information in a <STYLE>-Tag.
