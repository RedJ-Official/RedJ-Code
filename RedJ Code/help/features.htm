Features

**File Stuff**
 - Create/Open/Save files
 - Recent files list
 - Export with syntax highlighting as HTML or RTF
 - Print files
 - Show file properties and rename/change encoding without having to close the file

**Editing**
 - Multilevel Undo/Redo
 - Cut/Copy/Paste
 - Drag-Dropping inside and from outside the application
 - Select All
 - Find/Replace/Insert Line Prefix
 - Go To Line
 - Increase/Decrease/Automatic Indentation
 - (Un)comment code
 - Move Selected Line Up/Down
 - Quickly Create XML/HTML Tags
 - Advanced String Manipulation (Base64 encode/decode, URL encode/decode, etc.)
 - Case conversion (upper, lower, title, sentence, inverted, random)
 - Insert date/time
 - Insert predefined code snippets

**Features**
 - Word Wrap (with or w/o auto indent)
 - Line Numbers
 - Folding Blocks and Folding Lines
 - Highlight current line/current word occurrances/bracket pairs
 - Show/hide invisible chars
 - Auto Indent new lines
 - Auto complete bracket pairs
 - Virtual Space
 - Document map
 - Ruler
 - Autocomplete Menu (for some languages)
 - Bookmarks
 - Macros

**Supported Languages**
 - C#
 - CMD Batch
 - HTML
 - Java
 - JavaScript
 - JSON
 - Lua
 - PHP
 - Python
 - SQL (for SQL Server and MySQL)
 - Visual Basic
 - XML

**Additional Tools**
 - Terminal (cmd.exe wrapper but inside the editor)
 - Explorer (explorer.exe embedded but inside the editor)
 - MergeDiff
 - GUID Generator
