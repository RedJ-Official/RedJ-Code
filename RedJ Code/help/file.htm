Basic File Commands

  • New: Creates a new empty file.
  • Open: Opens a file from the local file system.
  • Recent: Open a recently edited file.
  • Save: Saves the current file.
  • Save As: Saves a copy of the current file.
  • Export: Export the current file with syntax highlighting. (Show Details)
  • Reload: Reload the current file from the disk.
  • Close: Closes the current file.
  • Properties: Shows the properties of the current file.
  • Print: Print the current file wich syntax highlighting.
