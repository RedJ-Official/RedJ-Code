Basic Editing

  • Undo: Undo the last action.
  • Redo: Redo the last undone action.
  • Cut: Move the selected text to the clipboard.
  • Copy: Copy the selected text to the clipboard.
  • Paste: Insert text from the clipboard at the caret position.
  • Select All: Mark all text as selected.
  • Increase Indent: Increase the indentation of the selected text.
  • Decrease Indent: Decrease the indentation of the selected text.
  • Auto Indent: Automatically adjust the indentation of the selected text.
  • Comment: Insert the comment prefix for the selected text.
  • Uncomment: Remove the comment prefix from the selected text.
  • Upper Case: Convert the selected text to upper case.
  • Lower Case: Convert the selected text to lower case.
  • Title Case: Convert the selected text to lower case, except for the first letter of each word, wich will be converted to upper case.
  • Sentence Case: Convert the selected text to lower case, except for the first letter of each sentence, wich will be converted to upper case.
    • Invert Case: Convert the selected text to upper to lower and from lower to upper case.
  • Random Case: Randomly convert the selected text to upper or lower case.
